* 파이썬 가상환경 생성 방법 
터미널창 열고 -> 터미널 명령어 python -m venv "가상환경이름" 입력 및 엔터 
예) python -m venv chatbot_env
가상환경 폴더 "chatbot_env" 안에는
앞으로 가상환경에 설치할 모든 라이브러리 소스코드가
아래처럼 저장된다. (include, Lib, Scripts 등등...)

* 파이썬 가상환경 활성화 방법
터미널창 열고 -> 터미널 명령어 "가상환경이름"\Scripts\activate.bat 입력 및 엔터
예) chatbot_env\Scripts\activate.bat
위의 명령어가 뜻하는 것은 새로 생성한 가상환경 "chatbot_env" 안에 폴더 Scripts 안에 activate.bat 파일을 실행한다는 뜻이다.

가상환경이 활성화 되면 아래처럼 터미널 입력창 맨 왼쪽에 (가상환경이름)이 생성된다.
(예) (chatbot_env) D:\minjae\testKakaoChatbot\chatbot>

가상환경이 활성화 되면 이제 터미널 창 안에서 설치하는 모든 라이브러리는 해당 가상 환경 안에 설치가 된다.

* 파이썬 스크립트 파일(lambda_function.py)과 새로 설치한 가상환경(chatbot_env) 연결 방법 
1. VSCode 상단 리본 탭 "View" 클릭 -> 컨텍스트 메뉴 "Command Palette..." 클릭 
2. Python: Select Interpreter 검색 및 클릭
3. 방금 전에 설치했던 가상환경 "Python 3.11.4 ('chatbot_env') .\chatbot_env\Scripts\python.exe" 클릭
4. 해당 파이썬 스크립트 파일(lambda_function.py)은 방금 전 생성한 가상환경 "chatbot_env"와 연결 되어 해당 가상환경 안에서 작업 가능하다.

* 파이썬 스크립트 파일(lambda_function.py) 실행 방법
터미널창 열고 -> 터미널 명령어 python lambda_function.py 입력 및 엔터
