AutoCAD 2024 버전 Language Pack 한국어 설치 방법

AutoCAD 2024 한글판 제품 설치방법을 안내 드립니다.

1. 먼저 manage.autodesk.com 홈페이지로 접속합니다.
2. 홈페이지에서 흰색 로그인 버튼을 클릭합니다.
3. 먼저 아이디를 입력하고, "다음" 버튼을 클릭합니다.
4. 비밀번호를 입력하고, "로그인" 버튼을 클릭합니다.
5. 2차 인증을 진행하고 코드입력 버튼을 클릭합니다.
6. manage.autodesk.com 포털창에서 "제품 및 서비스" 탭을 클릭합니다.
7. AutoCAD 항목을 찾고, 필요한 제품의 버전 및 언어를 선택합니다. 검정색 "설치" 버튼을 클릭하고 "동의" 버튼을 클릭합니다.
8. 다운로드된 웹브라우저 설치파일을 실행합니다.
9. 설치 구성요소가 준비될때까지 대기하십시오.

*만약 정상적으로 수행하기 힘드시다면 수동 설치파일 다운로드 링크를 안내드립니다.*
해당 링크에 게시된 모든 설치파일을 다운로드 하고, 1번 파일을 실행합니다.
1번 파일
https://efulfillment.autodesk.com/NetSWDLD/2024/ACD/2BBD2191-BB12-3557-8D70-4B33544CA3E0/SFX/AutoCAD_2024_Korean_Win_64bit_dlm_001_002.sfx.exe?authparam=1748764726_9956abc7a488a12e5b9bebf704065282
2번 파일
https://efulfillment.autodesk.com/NetSWDLD/2024/ACD/2BBD2191-BB12-3557-8D70-4B33544CA3E0/SFX/AutoCAD_2024_Korean_Win_64bit_dlm_002_002.sfx.exe?authparam=1748764726_e5e98c777e8f53ead833fb9b3772c575

10. 설치 UI가 활성화되면 "다음"을 클릭합니다.
11. *유의사항* 설치위치는 C:\Program Files\Autodesk에 고정합니다 변경하지 마십시오
12. "설치"를 클릭합니다.
13. 설치가 100% 전부 다 진행될 때까지 기다립니다.
14. 마지막으로 재시작이 필요할 경우 "다시 시작" 버튼을 클릭합니다.
15. 재시작 후 AutoCAD를 실행합니다.

* 라이센스 활성화 *

- Autodesk ID로 로그인을 클릭합니다.
- 인터넷 창에서 Autodesk 로그인 창이 뜨게 되며, Autodesk ID를 입력합니다.
- 다음을 클릭합니다.
- 비밀번호를 입력합니다.
- 로그인을 클릭합니다.
- 2차 인증을 진행하고 코드입력 버튼을 클릭합니다.
- 로그인이 완료되면, 제품으로 이동을 클릭합니다.
- 라이센스 정보가 리디렉션 되면서 인증이 완료됩니다. AutoCAD의 라이선스 인증이 이루어집니다.  
- 만약 이전 버전의 AutoCAD를 설치한 상태에서 새로운 버전을 설치할 경우, 이전 버전의 AutoCAD 설정을 새 버전으로 마이그레이션 할 수 있습니다.

RevitBOX 2024 버전 설치 방법

RevitBOX 2024 제품 설치방법을 안내 드립니다.

1. 설치파일 다운로드 링크
Google Drive 혹은 HubSpot 중 호환되는 링크주소를 클릭하여 설치파일을 다운로드 합니다.
(Google Drive 설치파일 다운로드 링크)
https://drive.google.com/file/d/1vJi9Hjxw7r0JHNuR3nsmLcV7AEs6Pp9S/view?usp=sharing
(HubSpot 설치파일 다운로드 링크)
https://43834191.fs1.hubspotusercontent-na1.net/hubfs/43834191/Chatbot/1.%20RevitBOX/2024/Imagine%20Builder%20RevitBox%202024_x64_20240729_AABIM.zip

2. 다운로드된 ZIP파일의 압축을 먼저 풉니다.
3. 압축이 풀린 .EXE 설치파일을 실행합니다.
4. 다음을 클릭합니다.
5. 사용권 계약에 동의함을 클릭합니다.
6. 설치가 100% 전부 다 진행될 때까지 기다립니다.
7. 마침을 클릭합니다.
8. Revit 2024를 실행합니다.

라이선스 인증

BOX 제품에 대한 라이선스 인증의 팝업이 발생합니다.
- ㈜상상진화가 보낸 이메일 편지함을 확인하시면, 고객번호와 제품번호에 대한 자세한 정보를 확인할 수 있습니다.
- 고객번호는 일반적으로 사업자 등록 번호이며, 이메일을 통하여 정확히 확인할 수 있습니다.
- 제품번호는 이메일을 통하여 정확히 확인할 수 있습니다.
- 만약 고객번호와 제품번호를 확인할 수 없다면, 상상플렉스 커뮤니티(기술지원 문의)로 문의 부탁드립니다.
- 고객번호와 제품번호를 다 입력했다면, 온라인 인증을 클릭합니다.
- 고객번호와 제품번호가 일치하면, 하단의 온라인 인증 란이 활성화됩니다.
- 회사명, 이름, 이메일, 연락처를 입력하고, 우측 하단에 인증을 클릭합니다.
- 인증이 완료되었습니다. 메시지가 발생합니다. 확인을 클릭합니다.
- 이제 RevitBOX를 활용할 수 있습니다.

Autodesk 계정 비밀번호 분실

Autodesk 계정 비밀번호를 분실했을 시 조치방법을 안내 드립니다.

1. 비밀번호 입력창에서 “잊으셨습니까?” 항목을 클릭합니다.

2. "암호 재설정: 받은 편지함에서 암호 재설정 방법을 설명하는 전자 메일 찾기" 해당 창이 활성화된 것을 확인합니다.

3. Autodesk 아이디에 입력된 이메일로 들어가 받은 편지함을 확인합니다.

4. Autodesk에서 “암호 재설정”이라는 제목으로 보내진 이메일을 엽니다. (간혹 스팸 메일 함에 보관되어 있을 수 있으니 유의 부탁드립니다.)

5. 이메일 상에 “암호 재설정” (RESET PASSWORD) 버튼을 클릭합니다.

6. 암호 재설정 인터넷 창이 새로 활성화되며, 새로 설정할 암호를 입력합니다.

7. 암호 확인 란에 다시 새로 설정할 암호를 입력합니다.

8. 암호 재설정을 클릭합니다.

9. 비밀번호가 새로 설정되어, 이제 https://manage.autodesk.com 으로 접속하여 설정된 비밀번호로 다시 로그인을 진행합니다.
